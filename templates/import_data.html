Upload CSV File
Upload a CSV file containing equipment data. The file should match the expected format shown below.
Import Instructions
Before importing:
  • Ensure your CSV file has the correct column headers
  • Date fields should be in YYYY-MM-DD format
  • Boolean fields should use TRUE/FALSE values
  • Numeric fields should contain only numbers
Import process:
  1. Select your CSV file using the file input above
  2. Click "Import Data" to begin the process
  3. The system will validate and import your data
  4. You'll receive a confirmation message with the number of records imported
Important: This will add new equipment records to the database. Make sure your data is properly formatted to avoid errors.

Personnel Matching & Creation:
  • ID-based matching: If personnel ID is provided, it will match exactly to existing Personnel records
  • New personnel: If ID is blank but name is provided, a new Personnel record will be created
  • Automatic roles: Personnel are automatically assigned roles (contact, supervisor, physician) based on the CSV fields they appear in
  • Email handling: Emails are only set for new personnel - existing personnel emails are not updated from equipment imports
  • Human readability: Names and emails are included for CSV readability but matching is done by ID
