Import Instructions
Before importing:
  • Ensure your CSV file has the correct column headers
  • Date fields should be in YYYY-MM-DD format
  • Boolean fields should use TRUE/FALSE values
  • Numeric fields should contain only numbers
Import process:
  1. Select your CSV file using the file input above
  2. Click "Import Data" to begin the process
  3. The system will validate and import your data
  4. You'll receive a confirmation message with the number of records imported
Important: This will add new equipment records to the database. Make sure your data is properly formatted to avoid errors.
